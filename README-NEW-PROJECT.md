# Новый проект с блокнотом благодарностей

Краткая инструкция, как сделать для заказчика сайт по образцу love и подключить его к общей БД на derbugov.ru.

## 1. Скопировать проект

Скопируй папку `love` (или репозиторий с GitHub) в новый каталог/репо. Нужны:

- `index.html`
- папки `css/`, `js/`, `img/` (если есть)
- папку `api/` копировать **не нужно** — API один на derbugov.ru

## 2. Задать атрибуты формы блокнота

В `index.html` найди форму с `id="gratitudeForm"` и выставь:

| Атрибут | Значение | Описание |
|--------|----------|----------|
| `data-api-base` | `https://derbugov.ru/love/api` | Адрес API (общий для всех проектов) |
| `data-project` | **уникальный идентификатор** | Один проект = один блокнот в БД. Только латиница, цифры, дефис, подчёркивание (например `ivan-maria`, `anna-dmitry`) |

Пример:

```html
<form id="gratitudeForm" action="#" method="post"
    data-api-base="https://derbugov.ru/love/api"
    data-project="ivan-maria">
```

У каждого заказчика — свой `data-project`, тогда записи в БД не смешиваются.

## 3. Подстроить под заказчика

- Тексты, фото, стили — как нужно для их сайта.
- Имена в блокноте («От кого: Сергей / Оля») — замени на имена пары в полях `<legend>`, `<label>` и значениях `value="sergey"` / `value="olya"` (или свои, но тогда нужно будет поправить отображение в API, где задаётся `author_name`).

## 4. Публикация

- **GitHub Pages:** залей проект в репозиторий, включи Pages — блокнот будет работать через API на derbugov.ru.
- **Другой хостинг:** выложи только статику (HTML/CSS/JS). Папка `api/` на хостинге не нужна.

## БД на derbugov.ru

- Один раз выполнена миграция — в таблице `gratitude` есть колонка `project` (см. `api/add-project-column.sql`, если настраиваешь с нуля).
- Все проекты пишут в одну таблицу; разделение — по полю `project` (оно совпадает с `data-project` на сайте).

### Переименование проекта (love → sergey-olya)

Если сменил `data-project` с `love` на `sergey-olya` и хочешь, чтобы старые записи тоже отображались, выполни на хостинге один раз:

```sql
UPDATE gratitude SET project = 'sergey-olya' WHERE project = 'love';
```

Файл с этим запросом: `api/migrate-love-to-sergey-olya.sql`.
